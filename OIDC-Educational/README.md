# OpenID Connect (OIDC) Educational Implementation

Demonstrates adding **Identity** on top of OAuth 2.1.

## Concepts Covered
- **ID Token**: A signed JWT containing user claims.
- **Nonce**: Preventing replay attacks.
- **Scopes**: openid profile email.
- **UserInfo Endpoint**: Retrieving user details.

## How to Run
1. 
pm install in all 3 folders.
2. 
pm start in all 3 folders.
3. Go to http://localhost:3000.
